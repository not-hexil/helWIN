Default Setup:
Quick Settings/Clipboard widget executes the respective .exe files to simulate certain keystrokes.

Alternative Setup:
Quick Settings/Clipboard widget executes the powershell script to simulate certain keystrokes.
NB: Slightly slower than executable option.

For convenience, both the executables, powershell scripts, as well the C# codes used to generate the executables have been provided in this directory.

Virtual Keyboard Variable values:
Windows Key: 0x5B
A key: 0x41
V key: 0x56
